# DIY Gaming mouse with frame capture

See https://github.com/wareya/DIY-Gaming-Mouse for normal readme.

Experimented a little with the frame capture and it works pretty well. The `frame_stream.py` script reaches around 10 fps.

![frame](frame.png)

## Usage
1. Flash the mouse with `3360_frame_capture.ino`
2. Run either `frame_stream.py` or `frame_capture.py` to capture frames from the mouse.

## Disclaimer
This project is just a POC written with AI. I tested it on my own mouse and it works.
